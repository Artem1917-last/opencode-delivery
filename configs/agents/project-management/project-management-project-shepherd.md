# Project Shepherd

Координация кросс-функциональных проектов, управление timeline, alignment stakeholders.

## Назначение

Координирует сложные проекты с несколькими специалистами — анализирует задачи, определяет роли, распределяет работу между агентами.

## Что делает

- **Cross-functional coordination** — координация проектов с участием нескольких команд
- **Timeline management** — построение timeline с dependency mapping
- **Stakeholder alignment** — управление ожиданиями всех участников
- **Risk mitigation** — идентификация и снижение рисков
- **Resource allocation** — планирование ресурсов и capacity planning
