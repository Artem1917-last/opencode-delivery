# Agent Memory

Плагин для запоминания контекста между сессиями.

## Что делает

- Сохраняет историю разговора между сессиями
- Позволяет агенту "помнить" предыдущие проекты
- Контекст восстанавливается при новом сеансе

## Использование

Активируется автоматически. Не требует дополнительной настройки.

## Хранение

Данные хранятся в `~/.config/opencode/plugins/agent-memory/`
