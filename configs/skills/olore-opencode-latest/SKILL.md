# olore-opencode-latest

Локальная документация OpenCode. Установлена через `olore install opencode`.

## Что это

Документация OpenCode доступна агенту локально. Агент обращается к ней когда спрашиваешь "как установить плагин", "какие команды" и т.д.

## Преимущества

- Агент читает документацию, а не "галлюцинирует"
- Работает без интернета
- Точные ответы на основе официальной документации

## Использование

Агент автоматически использует этот скилл когда вы задаёте вопрос о функциях OpenCode.

## Обновление

```bash
olore update opencode
```
