# mcp-finder

Поиск MCP серверов для любой задачи.

## Что делает

- Natural language search по 6,700+ MCP серверам
- Рекомендует оптимальный сервер для задачи
- GitHub stars и рейтинг

## Использование

```
Пользователь: "Мне нужен MCP для работы с базой данных Postgres"
Агент: "Сейчас найду подходящий MCP сервер"
↓
mcp-finder: "Нашёл postgresql-mcp (GitHub stars: 2.3k), postgres-mcp (0.8k)"
Агент: Устанавливает и настраивает выбранный сервер
```

## Индексированные данные

- 6,700+ MCP серверов
- Natural language description search
- REST endpoint + MCP tool

## Примечание

Данные отправляются на api.rhdxm.com (без sensitive данных)
